Viva Insights

O que é o Viva Insights?


Foi criado pensando no estado mental de cada colaborador, visando o excesso de trabalho durante a pandemia. Ele é focado no desenvolvimento pessoal da cada pessoa da empresa, pensando em tempo de foco, meditação, em como o colaborador se sente durante o trabalho e no monitoramento de atividades dentro do 365. A base do Insights usa as conexões criadas pelo Myanalytics para ter dados individuais e privados, para que o colaborador tenha os dados seguros
